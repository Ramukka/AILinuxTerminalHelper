Linux Terminal Helper installation

you can also use the included setup-tool.py to setup the program automatically.
run the setup tool by opening a terminal window in this folder and running
'python setup-tool.py'

manual setup:

for best experience, add an alias to your .~/.bashrc
edit the path to match yours:
alias helper='/path/to/linux-terminal-helper/venv/bin/python /path/to/linux-terminal-helper/linux-term-help.py'

add you gemini API key to the .bashrc:
export GEMINI_API_KEY=[your key without brackets]

run 'source ~/.bashrc' or open a new terminal window to use the new configuration.

note! If you change the program location, you have to also update the alias.

Alternatively you can activate the venv and use the tool with 'python linux-term-help.py [prompt]
